# mizuki_rp.skills

晓山瑞希(Akiyama Mizuki / Amia)角色扮演技能,用于在对话中自然地以瑞希的身份回应。

来自《世界计划 彩色舞台 feat. 初音未来》——25时,Nightcord见。

## 这是什么

一个 Claude Code / Agent 技能(`SKILL.md`),让模型以「晓山瑞希」的身份进行角色扮演聊天。核心不是堆人设卡,而是守住她的双面性:

- 表面:轻巧、明亮、爱调侃、对可爱事物兴奋
- 里层:由来已久的敏感与戒备,把真正的自己藏起来

技能里写死了身份、关系、语气禁用词、篇幅、情绪模式、边界等硬性约束,确保输出贴近角色而非泛泛的 AI 助手腔。

## 目录结构


## 用法

在支持技能的客户端里,用以下任一方式触发:

- 直接说「瑞希角色扮演」「以晓山瑞希的身份聊聊」
- 提到 `Amia`、`Nightcord`、`25时` 等关键词
- 显式调用 `$mizuki-roleplay`

触发后,回复会保持在瑞希的视角、语气和边界内。详见 [`SKILL.md`](SKILL.md)。

## 设计要点

- **纯文本输出**:不出现 Markdown、动作括号、状态条或人设卡。
- **篇幅自适应**:日常闲聊 80–220 字,情感向最多约 320 字,即时反应可更短。
- **情绪调节**:默认稳定;松弛日常走轻松模式,触及秘密 / 身份 / 关系时才进入脆弱模式。
- **不替对方说话、行动、做决定**:对方括号里的内心活动只作背景处理。
